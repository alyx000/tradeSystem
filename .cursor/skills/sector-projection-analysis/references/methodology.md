# 板块推演术方法论提炼

本文件是对《0524板块推演术》的结构化提炼，供 agent 在板块分析时按需读取。

## 1. 目标

板块推演的核心不是“精准预测”，而是做两件事：

1. 看清热门板块大致处于哪个大级别阶段。
2. 找出当下哪个板块的节奏，刚好匹配大势需要的节奏，因此更可能成为最强或最确定的板块。

重点是**横向比较后挑最优**，而不是把某一个板块判断到极致精准。

## 2. 两套阶段

### 小级别情绪周期

- 启动
- 发酵
- 高潮
- 分歧
- 衰退

这里不要过度纠缠细节，更重要的是看**连接点是加强还是减弱**。

### 大级别粗放阶段

- 将成龙
- 主升
- 震荡
- 二波
- 衰退

其中：

- `震荡 / 二波 / 衰退` 可视为“充分演绎阶段”

## 3. 板块结构优先级

优先关注这些板块：

1. 当晚日 K 趋势最强，且短线也走强的大主流板块
2. 近期活跃、调整过、趋势未坏、出现止跌或逆势信号的板块
3. 新启动、逻辑不错、个股先形成趋势、但联动性还不大的板块
4. 新启动且联动很强的板块
5. 超跌后出现止跌信号、且具备逻辑的板块

独立个股也可以被视作一种“风格化板块”。

## 4. 强势板块的节奏判断

### 主动定性

对每个强势板块都要做一个主动定性：

- 它是不是潜在的大级别主线题材？
- 它是更依赖大势和逻辑，还是更依赖自身走势强度？

越初期，越依赖三位一体：

- 大势是否支持
- 板块逻辑/驱动是否成立

越后期，越依赖走势：

- 是否分歧不死
- 是否低预期却走强

### 阶段对应的合理模式

- `将成龙`：优先最票，强则第一时间买，弱则等确认
- `主升`：优先第一个能买到的买点，不必等太多盘口确认
- `震荡`：优先分歧低吸，如箱体、均线、首阴

详细的最票识别方法论与阶段匹配规则，见 [leader-identification.md](leader-identification.md)。

### 机会优先级

- 将成龙：先行启动的最票
- 主升初期：最强票、20cm 套利、中军波段
- 主升后期：中军低吸、性价比套利低吸
- 充分演绎：活口、新日内龙、支线挖掘、活跃票不走 A

## 5. 回流判断

主要看横向比较与节奏匹配。

两个高频原则：

1. 最强趋势板块短期转弱时，优先考虑最近活跃、最好调整了 2-3 天的板块。
2. 某启动板块快速高潮，次日转弱分化但强度还在，且盘面没有更强方向时，存在次日再回流加强、形成主线预期的可能。

除此之外，才去补充“三位一体”的感觉判断，尤其是**走势之外的理由**。

## 6. 如何看充分演绎板块是否结束

充分演绎通常有两个特征：

- 反复爆炒过，赚钱效应活跃
- 逻辑被市场广泛认可，几乎所有人都知道

此时观察重点：

1. 高位辨识度票在震荡后期是否还很硬
2. 板块里是否还有绝对辨识度票继续创新高
3. 是否同时出现新票走强

如果有“老核心继续硬 + 新票走强”，通常要多看一眼，可能意味着还有新一波弹性机会。

## 7. 逻辑审美

初期看：

- 利好是不是明确增量
- 增量空间有多大
- 增量能否落地，何时落地
- 板块当前所处阶段是否还有性价比
- 谁主导变化
- 逻辑对应的方向是否足够具体、容量是否够大

中期更看：

- 走势强度
- 持续性

一个逻辑真强，启动后通常会反复走，分歧后的反弹高点应越来越高。若逻辑吹得很大，但涨一段后反弹强度明显不行，要提高警惕。

## 8. 使用提醒

- 不追求个案绝对对错，重点是让自己提前形成定性，找到合理买点与观察点。
- 判断错，很多时候只是错过或卖飞，真正有损失的是“错误参与且无应对”。
- 短线交易里最有价值的能力，不是精准预言某个板块，而是在一堆板块中都能做大致判断，然后挑出你认为最好的那个。
