# 最票识别方法论

本文件是对《三位一体教程》中"最票"概念的结构化提炼，供 Agent 在复盘第 5 步识别最票时参考。

## 1. 最票定义

最票 = 个股在板块中某一个属性下的第一名。

- 第一名越清楚越好，越模糊越不好
- 选择标准：一定要简单，不要勉强
- 有引领属性的最，往往一眼就能看出来

## 2. 最的属性分类

### 引领类（价值最高）

- **走势引领**：分时主动走强，带动板块方向。在拐点/启动预期节点，主动带动板块走强
- **最先板**：板块内最早涨停
- **最高标**：连板高度最高

### 容量类

- **容量最大**：成交额/市值最大。风格化偏好大票时价值高
- **中军**：板块成交主力

### 逻辑类

- **基本面最正宗**：行业龙头地位，业绩/逻辑最直接

### 情绪类

- **连板最高**：连板数第一名
- **风格化最强**：匹配当前阶段审美偏好

**有价值的排序**：走势引领 > 最先板 > 最高标 > 风格化最强

## 3. 引领性最票的多因子判断

基于教程中海光信息案例提炼的极品引领最判断框架：

| 因子 | 含义 | 示例 |
|------|------|------|
| 逻辑正宗度 | 是否行业核心、逻辑最直接 | 海光信息 = 国产AI芯片龙头 |
| 辨识度 | 一眼可辨还是需要分析 | 极高 → 有价值 |
| 风格化匹配 | 符合当前审美偏好 | 大票+趋势 |
| 预期差节点 | 在低预期时主动走强 | 板块启动日低位主动走强 |
| 新最 | 以前没当过龙头，首次引领 | 首次在该板块引领 |

**判定规则**：
- 3+ 因子同时满足 → 极品引领最，可能是阶段龙头（不仅当天）
- 1-2 因子 → 普通最票，当日有效

## 4. 新最的识别信号

新最 = 以前没有当过龙头，当天在板块里分时最直接引领板块的个股。

识别要点：
- 从低位主动分时一直在走强
- 在预期差节点（板块启动或轮动）出现
- 逻辑正宗 + 辨识度高 + 符合风格化 → 可能不仅是当天龙头，而是接下来一段时间的龙头

## 5. 启动龙头两种极品模式

- **盘中半路引领**：开盘不久，半路介入极品引领票。关键是提升对"引领"的理解——在拐点预期、启动预期节点，主动带动板块走强
- **尾盘强分**：尾盘出现强分歧买点。强成龙 + 强分 = 高确定性

## 6. 板块阶段 × 应关注的最票类型

| 板块阶段 | 优先关注 | 模式 |
|---------|---------|------|
| 将成龙 | 先行启动的引领最 | 盘中半路 / 尾盘强分 |
| 主升初期 | 最强票 + 20cm套利 + 中军波段 | 第一个能买到的买点 |
| 主升后期 | 中军低吸 + 性价比套利低吸 | 分歧低吸 / 均线 / 首阴 |
| 充分演绎 | 活口 / 新日内龙 / 支线挖掘 | 活跃票不走A |

## 7. 老龙头 vs 新最的选择

核心原则：不同阶段聚焦不同龙头票，聚焦当时那个阶段更有发展力的龙头票。

- 充分演绎的老龙头：再怎么说都是节奏波动、大震荡，指望再来一波主升难度很大
- 新最出现时：关注新最的引领性和板块节奏的匹配度
- 高难度思考：如何平衡干老龙头的活跃节奏波动（滚动）vs 干新最的启动

## 8. 首阴价值判断

- 大势与板块都处于上升趋势初期 → 首阴越有价值
- 大势与板块充分演绎程度越高 → 首阴越没有价值

## 9. 使用约束

- Agent 推荐最票时必须标注 `[判断]`，不得伪装为 `[事实]`
- 客观可查的数据（涨停数、成交额排名）标注 `[事实]`
- 用户拍板后以用户结论为准，不反复质疑
- 数据不足时提示缺失，不硬凑推荐
